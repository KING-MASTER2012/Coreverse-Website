# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Coreverse/coreverse-website.git
```

Move into the project:

```bash
cd coreverse-website
```

Install dependencies:

```bash
npm install
```

Create:

```text
.env.local
```

Example:

```env
NEXT_PUBLIC_SUPABASE_URL=

NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

Start development:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
